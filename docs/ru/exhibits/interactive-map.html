Геопространственные данные · оккупированный Мариуполь

Каждое документированное дело — на карте.

Четыре опубликованных дела-исследования (9 нанесённых точек) на фоне полной геокодированной базы изъятий собственности проекта (2,683 документированных объекта) и реестра снесённых зданий из 470 объектов. Нажмите на любой маркер для краткого описания, источников и ссылки на полный экспонат, если он существует. Слои переключаются ниже.

Эта карта показывает только геокодированные, документированные объекты. Большинство точек (1,774 из 2,683 в слое базы) подтверждены на уровне, пригодном для юридического требования (достоверность ≥0,80 по стандарту источников этого проекта); 13 не достигают этого порога и помечены «ожидает проверки» во всплывающем окне; 896 несут независимое перекрёстное подтверждение, которому пока не присвоена оценка достоверности («неоценено») и которые не следует считать юридически значимыми, пока это не сделано. Ничего из этого не скрыто — проверяйте всплывающее окно точки перед тем, как ссылаться на неё.

Источники. Точки дел-исследований привязаны к базе PostgreSQL проекта (хешировано по цепочке хранения), где геокодировано; два здания, пока не геокодированных в самой базе (просп. Строителей 108; ул. Сеченова 54), размещены через независимый поиск адреса OpenStreetMap/Nominatim и помечены «приблизительно» во всплывающем окне — не юридически значимо, только для ориентира. Дело о судебном реестре (28 судей, 2,666 решений) намеренно не имеет точки здесь — его центральный вывод состоит в том, что ни в одной из этих 2,666 записей не указан адрес; см. полный экспонат. Слой снесённых зданий. 470 зданий в реестре снесённых; для 20 из них найден соответствующий ордер на перераспределение — модель подмены адреса снос→восстановление, документированная в деле Нахимова 82. Всплывающие окна базы и топонимика. Украинское название после реформы декоммунизации 2016 года считается каноническим для довоенного периода; латинская транслитерация всегда строится от него, а не от русского написания, даже если в деле имеется только адрес из оккупационного источника. Там, где запись топонимов этого проекта показывает, что оккупант вернул улице её довоенное советское/коммунистическое название (это верно для каждого документированного переименования на сегодня — например, проспект Миру, сам являвшийся заменой 2016 года советского проспекта Ленина, был возвращён оккупантом к имени Ленина), всплывающее окно показывает оба варианта: украинское название и восстановленное советское, каждое с меткой. Каждая точка также несёт категорию требования RD4U и наиболее обоснованную запись для неё — датированное постановление жизненного цикла, если оно есть, иначе наиболее сильное независимое подтверждение в деле (оценка ущерба, спутник, медиа жителей и т.д.), явно обозначенное как таковое. Объекты без категории RD4U, без постановления, без подтверждения и без судебного дела вообще исключены из этого слоя, а не показаны без какого-либо обоснования.
